# Particles game of life

![Demo](out.gif)

## How to start

Clone the project
```
git clone https://github.com/Alianshark/Particles-game-of-life.git
```

Change dir to the proejct folder
```
cd Particles-game-of-life
```

Run any server inside the project folder, for example
```
npx http-server .
```
